================================================================================
THE BOOKING WORKFLOW
================================================================================

1. STUDENT LOGS IN
   - Browses available rooms/beds
   - Selects a room with available beds
   - Submits booking request

2. MANAGER REVIEWS
   - Sees pending booking request
   - Reviews student details
   - Approves or rejects

3. PAYMENT & ACTIVATION
   - Student receives notification
   - Student processes payment
   - System allocates bed
   - Student can check in

4. OCCUPANCY & MANAGEMENT
   - Manager tracks occupancy
   - Student can file complaints
   - Complaints can be resolved
   - Booking cycle completes
