# Referral System Addon

This addon enables referral features in the hostel system:

- Student referral dashboard
- Partner referral portal
- Commission tracking
- Payout requests
- Referral settings (notifications and referral popup)

## Installation

1. Go to Admin panel -> System -> Addons.
2. Upload the provided ZIP file.
3. Activate `referral-system`.

The package requires `addon.json` at ZIP root for discovery.
